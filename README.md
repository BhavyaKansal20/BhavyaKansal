## 📌 Overview

Welcome to the source code for my personal portfolio website! This repository contains a highly optimized, beautifully designed web application built with modern frontend technologies. It serves as my digital resume, showcasing my projects, research, and expertise in Artificial Intelligence and Machine Learning.

---

## ⚡ Tech Stack

This project is built using industry-standard tools for maximum performance and a premium user experience:

*   **Frontend Framework:** React 18
*   **Build Tool:** Vite (Lightning-fast HMR and optimized builds)
*   **Styling:** Tailwind CSS (Utility-first CSS) + CSS Variables
*   **UI Components:** Shadcn UI & Radix UI Primitives
*   **Animations:** Framer Motion (or native CSS animations via `tailwindcss-animate`)
*   **Routing:** React Router DOM
*   **Icons:** Lucide React

## 🛠️ Local Development

Want to run the portfolio locally? Follow these steps:

### 1. Clone the Repository
```bash
git clone https://github.com/BhavyaKansal20/BhavyaKansal.git
cd BhavyaKansal
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash
npm run dev
```
Open `http://localhost:5173` in your browser to view the site.
